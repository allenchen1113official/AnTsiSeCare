# AnTsiSeCare - 安心長照 APP 完整規劃設計

> 以苗栗縣為場域的台灣長照服務整合平台

---

## 一、專案概述

### 1.1 專案背景

台灣已正式進入超高齡社會，苗栗縣因人口外移、高齡化程度顯著，長照資源分散且城鄉差距大。現行長照2.0雖已建立A-B-C三級服務體系，但民眾仍面臨：
- 不知道去哪裡找長照資源
- 照顧者資訊碎片化、溝通困難
- 緊急狀況無法即時通報
- 跨機構間照護紀錄無法串接

**AnTsiSeCare**（安心照護）旨在以數位化平台整合苗栗縣長照資源，提供一站式服務，降低照顧者負擔，提升被照顧者生活品質。

### 1.2 命名意涵

| 名稱 | 意涵 |
|------|------|
| An（安） | 安心、安全 |
| Tsi（置） | 妥善安置、照護到位 |
| Se（勢） | 苗栗客語「勢」，意為有能力、有辦法 |
| Care | 照護、關懷 |

### 1.3 願景與目標

**願景**：讓苗栗縣每位長者都能獲得有尊嚴、有溫度的長照服務

**核心目標**：
1. 整合苗栗縣全縣長照資源地圖（A/B/C級單位）
2. 提供照顧者與被照顧者即時溝通橋梁
3. 建立個人化照護計畫管理系統
4. 落實緊急通報與安全守護機制
5. 支援客語介面（苗栗在地化）

├── 模組5：緊急通報系統
├── 模組6：健康監測儀表板

| 功能 | 說明 |
|------|------|
| 苗栗縣長照單位地圖 | A級（旗艦店）、B級（社區）、C級（巷弄）分層顯示 |
| 機構搜尋篩選 | 依鄉鎮市、服務類型、評鑑等級篩選 |
| 服務項目查詢 | 居家服務、日間照顧、交通接送、餐飲服務等 |
| 候補即時狀態 | 顯示各機構目前候補人數 |
| 路線導航 | 整合 Google Maps 導航 |
| 長照專線一鍵撥打 | 1966 及苗栗縣在地專線 |

**苗栗縣各鄉鎮對應資源**：
- 苗栗市、頭份市（都市型，資源較充足）
- 三義、南庄、獅潭（山地偏鄉，資源匱乏）
- 泰安鄉（原住民族長照專區）

## 四、苗栗縣在地化設計

### 4.1 語言支援
- **中文繁體**（主要）
- **客語語音**（苗栗四縣腔）：語音導覽、SOS 呼救語音
- **原住民族語**（泰雅語）：泰安、南庄山地鄉使用者

### 4.2 苗栗縣長照資源整合清單

| 類別 | 機構 | 服務範圍 |
|------|------|---------|
| A級旗艦店 | 苗栗縣各鄉鎮市 A 級單位 | 整體照顧管理 |
| B級複合服務 | 日照中心、居家服務 | 社區型服務 |
| C級巷弄站 | 社區關懷據點、村里辦公室 | 在地預防保健 |
| 山地原鄉 | 泰安、南庄、獅潭專責服務 | 偏鄉特殊服務 |
| 特殊需求 | 失智症共照中心 | 失智友善服務 |

### 4.3 偏鄉服務策略
- 離線模式（Offline Mode）：山區網路不穩定，基本功能可離線使用
- 簡訊備援：網路斷線時以 SMS 發送緊急通報
- 低頻寬優化：壓縮圖片、延遲非必要資料載入

## 五、法規遵循

### 5.1 個人資料保護法（個資法）
- 蒐集目的明確告知、取得同意書
- 健康資料（特種個資）加密儲存
- 使用者可申請刪除帳號及所有資料
- 資料存取日誌留存

### 5.2 長照2.0政策對應
- 對應衛福部長照給付及支付基準
- 照護計畫符合 BPI 評估格式
- 服務紀錄支援匯出 CMS 格式

### 5.3 無障礙設計規範
- 符合 WCAG 2.1 AA 等級
- 支援 iOS VoiceOver / Android TalkBack
- 最小字體 16px，可調整至 24px
