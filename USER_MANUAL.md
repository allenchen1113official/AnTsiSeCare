## 1. App 簡介

**安心照護（AnTsiSeCare）** 是一款專為台灣長期照顧（長照 2.0）體系設計的照護管理 App，特別支援**外籍移工照服員**（印尼語、越南語、泰語）與**本國照服員**（繁體中文）共同使用。

### 核心功能

| 功能 | 說明 |
|------|------|
| 📋 照護日誌 | 每日照護紀錄、生命徵象量測、語音輸入、AI 翻譯 |
| 🆘 SOS 緊急求救 | 一鍵發送 GPS 定位、通知家屬與照管員 |
| 💊 用藥管理 | 藥品排程、服藥確認、逾時提醒 |
| 🗺️ 長照資源 | 全台 22 縣市長照機構查詢（A/B/C 級） |
| 👨‍👩‍👧 家屬視角 | 遠端即時查看照護紀錄與異常警示 |

---

## 2. 使用者角色

App 依據角色提供不同的功能權限：

| 角色 | 說明 | 可用功能 |
|------|------|---------|
| **照服員**（caregiver） | 外籍或本國照服員 | 全部功能：日誌、SOS、藥品、地圖 |
| **家屬**（family） | 被照顧者的家庭成員 | 查看日誌、收警示通知、SOS 通報接收 |
| **照管員**（care_manager） | 長照機構管理人員 | 查看所有被照顧者資料、統計報表 |
| **被照顧者**（elder） | 長者本人 | 簡化介面、SOS 求救、藥品提醒 |

## 3. 首次登入與設定

### 3.1 手機號碼登入

1. 開啟 App，於登入畫面輸入**台灣手機號碼**（格式：`+886 9XX XXX XXX`）
2. 點擊「**傳送驗證碼**」
3. 輸入收到的 **6 位數 OTP 驗證碼**（有效期 5 分鐘）
4. 驗證成功後進入角色選擇

> **測試帳號**：`+886 912 345 678`，OTP：`123456`

### 3.2 選擇角色

- 首次登入需選擇身份：照服員 / 家屬 / 照管員 / 被照顧者
- 角色確認後可在「個人設定」中申請變更

### 3.3 選擇介面語言

支援 5 種語言，可依個人需求選擇：

| 語言 | 代碼 | 備註 |
|------|------|------|
| 繁體中文 | zh-TW | 預設語言 |
| 印尼語 | id | 附加禱告時間提醒 |
| 越南語 | vi | — |
| 泰語 | th | — |
| 英語 | en | — |

## 5. 照護日誌

### 5.1 新增照護紀錄

1. 點擊首頁「**📋 新增日誌**」或底部導覽「**日誌**」
2. 系統自動記錄**台北時間**（UTC+8）的當前時間戳

### 5.2 照護項目（12 項）

點擊每個項目圖示可切換狀態：

| 狀態 | 說明 |
|------|------|
| ✅ 正常（normal） | 照護項目已完成且無異常 |
| ⚠️ 異常（abnormal） | 有異常狀況，**必須填寫備註**（10 字以上） |
| ⏭️ 跳過（skipped） | 本次未執行 |
| ❌ 拒絕（refused） | 被照顧者拒絕配合 |

**12 個照護項目：**

```
🍽️ 飲食    💊 用藥    🚽 排泄    🛁 沐浴
🏃 活動    😴 睡眠    😊 情緒    🩹 傷口
💬 溝通    🦽 移位    🏠 家務    🧠 認知
```

### 5.3 生命徵象量測

點擊「**＋ 生命徵象**」展開輸入欄位：

| 項目 | 正常範圍 | 異常條件 |
|------|---------|---------|
| 血壓（收縮壓） | 90–140 mmHg | `< 90` 或 `> 140` |
| 血糖 | 70–180 mg/dL | `< 70`（低血糖）或 `> 180` |
| 體溫 | 36.0–37.5 °C | `< 36.0` 或 `> 37.5` |
| 心率 | 60–100 bpm | `< 60` 或 `> 100` |
| 血氧（SpO₂） | ≥ 95 % | `< 95` |
| 體重 | — | 手動記錄 |

> ⚠️ 任何指標超出正常範圍，系統自動標記**異常警示**並通知家屬。

### 5.4 語音輸入（印尼語 / 中文）

1. **長按**備註欄右側的麥克風圖示 🎙️
2. 對麥克風說話（支援 `id_ID` 印尼語、`zh_TW` 中文）
3. 放開後語音自動轉為文字

> 需先在系統設定允許麥克風權限。

### 5.5 AI 翻譯（印尼語 → 繁體中文）

1. 輸入**印尼語備註**後，點擊「**🤖 翻譯**」按鈕
2. App 透過 Claude AI（`claude-sonnet-4-6`）將照護文字翻譯為繁體中文
3. 翻譯結果顯示於備註下方，可編輯後儲存

> 需要網路連線。翻譯針對長照用語進行最佳化。

### 5.6 儲存日誌

- **有網路**：直接上傳至 Firebase Firestore
- **無網路**：自動儲存至本地 Hive 快取，網路恢復後自動同步

## 6. SOS 緊急求救

> ⚠️ **重要**：此功能僅用於真實緊急狀況，誤觸請立即取消。

### 6.1 啟動 SOS

1. 點擊底部導覽「**🆘 SOS**」或首頁「緊急求救」按鈕
2. 畫面中央顯示**紅色脈搏動畫**大按鈕
3. **點擊大按鈕**選擇緊急類型

### 6.2 緊急類型

| 類型 | 說明 |
|------|------|
| 🫀 突發疾病 | 心臟病、中風等急症 |
| 🤕 跌倒受傷 | 摔倒、骨折 |
| 😵 失去意識 | 昏迷、意識模糊 |
| 😮‍💨 呼吸困難 | 哮喘、窒息 |
| 🧠 情緒危機 | 嚴重躁動、自傷風險 |
| 🆘 其他緊急 | 無法歸類的緊急情況 |

### 6.3 觸發後自動執行

```
① 取得 GPS 座標（精確位置）
② 寫入 Firestore emergencyAlerts 集合
③ 推播通知 → 家屬 & 照管員
④ 本地震動警示
⑤ 顯示緊急電話快速撥號
```

### 6.4 緊急電話

| 電話 | 用途 |
|------|------|
| **119** | 救護車 / 消防局（緊急救護） |
| **1966** | 長照服務專線 |
| **1955** | 移工諮詢（提供印尼語服務）⭐ 外籍照服員優先 |
| **110** | 警察 |

## 7. 用藥管理

### 7.1 查看今日用藥排程

1. 點擊底部導覽「**💊 藥品**」
2. 畫面依時段（早 / 午 / 晚 / 睡前）顯示藥品清單
3. 每個時段顯示各藥品名稱、劑量、服藥時間

### 7.2 記錄服藥

- 點擊「**✅ 已服用**」：記錄該筆服藥時間至 `medicationLogs`
- 點擊「**⏭️ 跳過**」：記錄跳過原因

### 7.3 新增藥品

1. 點擊右上角「**＋ 新增藥品**」
2. 填寫：
   - 藥品名稱（中文 / 印尼語）
   - 劑量規格（例：amlodipine 5mg × 1 顆）
   - 服用頻率：每日一次 / 每日兩次 / 每日三次 / 需要時 / 自訂
   - 服藥時間（多個時段）
3. 點擊「**儲存**」

### 7.4 用藥頻率說明

| 頻率 | 中文 | 印尼語 |
|------|------|--------|
| daily_once | 每日一次 | Sekali sehari |
| daily_twice | 每日兩次 | Dua kali sehari |
| daily_three | 每日三次 | Tiga kali sehari |
| as_needed | 需要時服用 | Bila perlu |
| custom | 自訂 | Kustom |

## 8. 長照資源地圖

### 8.1 查詢資源

1. 點擊底部導覽「**🗺️ 地圖**」
2. 資料來源：衛福部長照服務資源開放資料（24 小時快取）

### 8.2 篩選方式

**第一層：縣市選擇**
- 下拉選單選擇全台 22 縣市（或「全台灣」顯示所有）

**第二層：鄉鎮篩選**
- 選定縣市後，下方出現該縣市鄉鎮 chips，可進一步篩選

**等級篩選 chips：**

| 等級 | 說明 |
|------|------|
| **A 級** | 整合服務中心（旗艦型，最完整） |
| **B 級** | 複合型服務中心 |
| **C 級** | 巷弄長照站（社區型） |

**關鍵字搜尋：**
- 可搜尋機構名稱、地址或鄉鎮名稱

### 8.3 機構詳細資訊

點擊機構卡片顯示底部彈窗：
- 機構名稱（中文 + 印尼語等級說明）
- 地址
- 電話（點擊直接撥打）
- 服務時間
- 提供的服務項目
- **導航按鈕**（開啟地圖 App 導航）

### 8.4 等級說明（中 / 印尼語對照）

| 等級 | 中文 | 印尼語 |
|------|------|--------|
| A 級 | 整合服務中心 | Pusat Layanan Terpadu |
| B 級 | 複合型服務中心 | Pusat Komprehensif |
| C 級 | 巷弄長照站 | Pos Perawatan |

### 8.5 重新整理資料

點擊右上角「**↺**」強制從衛福部重新下載最新資料。

## 10. 離線模式

App 支援完整離線操作：

### 10.1 離線時可用功能

| 功能 | 離線狀態 |
|------|---------|
| 新增照護日誌 | ✅ 儲存至本地 Hive |
| 查看歷史日誌 | ✅ 讀取本地快取 |
| SOS 緊急求救 | ✅ 本地記錄 + 震動（推播無效） |
| 用藥管理 | ✅ 本地操作 |
| 長照資源查詢 | ✅ 讀取 24 小時快取資料 |
| AI 翻譯 | ❌ 需要網路 |
| 推播通知 | ❌ 需要網路 |

### 10.2 資料同步

- 網路恢復後，所有本地暫存的照護日誌**自動上傳**至 Firestore
- 首頁橘色「**📴 離線**」徽章消失表示同步完成

### 10.3 本地快取儲存空間（Hive）

| 快取區 | 說明 |
|--------|------|
| `care_logs_pending` | 待上傳的照護日誌 |
| `user_cache` | 使用者基本資料（1 小時有效） |
| `app_settings` | App 設定偏好 |
| `ltc_resources` | 長照資源（24 小時有效） |
| `medications` | 藥品清單 |

## 12. 常見問題

**Q：OTP 驗證碼沒收到怎麼辦？**
> 等待 60 秒後點擊「重新傳送」。請確認號碼格式正確（需含國碼 +886）。

**Q：照護日誌上傳失敗怎麼辦？**
> 資料已自動儲存本地，請確認網路連線後 App 會自動重試上傳。首頁若無離線徽章即表示已同步。

**Q：SOS 誤觸怎麼取消？**
> 在緊急類型選擇畫面點擊「取消」即可。若通知已發送，請主動聯絡家屬說明誤觸。

**Q：如何切換介面語言？**
> 前往「個人設定」→ 點擊「🌐 語言」→ 選擇所需語言後重啟 App 生效。

**Q：長照資源資料多久更新一次？**
> 每 24 小時從衛福部開放資料自動更新。也可點擊地圖畫面右上角「↺」手動強制更新。

**Q：AI 翻譯是否免費？**
> App 內建 AI 翻譯功能由 Claude API 提供，翻譯費用由服務方承擔，使用者無需付費。

**Q：資料安全嗎？**
> 所有資料傳輸皆使用 HTTPS 加密。Firebase 安全規則確保每位使用者僅能存取自己或負責照顧的長者資料。
