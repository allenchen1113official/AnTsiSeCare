# AnTsiSeCare - 多國語言國際化設計（i18n）

> 場域：全台灣 ｜ 第一優先語言：印尼語（Bahasa Indonesia）

---

## 一、為何需要多語言支援？

### 1.1 全台移工照護現況

台灣各縣市有大量**印尼籍家庭看護工**（TKI / Tenaga Kerja Indonesia）在此照顧長者。根據勞動部統計：

| 指標 | 數據 |
|------|------|
| 全台印尼籍看護移工 | 約 25 萬人（2025年） |
| 全台外籍看護估計 | 約 8,000–12,000 人 |
| 印尼看護工比例 | 約佔外籍看護 65% |
| 主要語言障礙情境 | 照護記錄填寫、與家屬溝通、緊急通報 |

**核心痛點：**
- 印尼籍看護工難以閱讀中文照護指示與計畫
- 緊急狀況發生時，語言不通造成通報延誤
- 服務紀錄需用中文填寫，造成照服員壓力
- 家屬與看護工溝通困難，照護品質難以監督

### 1.2 多語言支援的價值

```
家庭照顧者（台灣）←──── AnTsiSeCare ────→ 印尼看護工
     中文介面                 ↕                印尼語介面
                      即時翻譯橋梁
                      照護計畫共享
                      日誌自動翻譯
```

## 三、印尼語（Bahasa Indonesia）重點設計

### 3.1 印尼看護工使用情境

#### 情境 A：服務到達打卡
```
中文（家屬看到）：   李蒂抵達  09:05，開始服務
印尼語（看護看到）：Listi tiba pukul 09:05, mulai layanan
```

#### 情境 B：照護日誌填寫
```
印尼語介面引導：
「Apa yang Anda lakukan hari ini?」（您今天做了什麼？）
✓ Membantu mandi（協助沐浴）
✓ Menyiapkan makan（準備餐食）
✓ Menemani jalan（陪同散步）
□ Fisioterapi（物理治療）
□ Minum obat（服藥協助）
```

#### 情境 C：緊急通報
```
SOS 按下後，印尼語語音：
「Darurat! Saya butuh bantuan!」（緊急！我需要幫助！）
自動傳送位置給家屬，中文通知：「看護工已觸發緊急通報」
```

#### 情境 D：照護計畫查閱
```
照管專員用中文建立計畫
↓ 自動翻譯至印尼語
看護工用印尼語查看每日照護清單（Daftar Perawatan Harian）
```

```

  zh-TW：2026年4月14日（週一）

  zh-TW：上午 9:00

數字格式：
  zh-TW：1,200 元
  id：   Rp1.200 → 在台灣脈絡下顯示 NT$1,200

  早上（06-11時）：Selamat pagi ☀️
  下午（11-15時）：Selamat siang
  傍晚（15-19時）：Selamat sore
  晚上（19-06時）：Selamat malam

宗教考量（印尼以伊斯蘭為主）：
  - 禱告提醒（選用功能）：Waktu Salat
  - 飲食記錄：標注清真（Halal）食物選項
```


### 4.4 i18next 初始化設定

```typescript
// src/i18n/index.ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import * as RNLocalize from 'react-native-localize';

// 語言資源匯入
import zhTW from '../locales/zh-TW';
import id from '../locales/id';
import en from '../locales/en';

const resources = {
  'zh-TW': { translation: zhTW },
  'id':    { translation: id },
  'en':    { translation: en },
};

// 偵測裝置語言
const getDeviceLanguage = (): string => {
  const locales = RNLocalize.getLocales();
  const supported = ['zh-TW', 'id', 'en'];
  for (const locale of locales) {
    const tag = locale.languageTag;
    if (supported.includes(tag)) return tag;
    // 印尼語裝置（id-ID → id）
    const lang = locale.languageCode;
    if (supported.includes(lang)) return lang;
  }
  return 'zh-TW'; // 預設繁體中文
};

i18n
  .use(initReactI18next)
  .init({
    resources,
    lng: getDeviceLanguage(),
    fallbackLng: 'zh-TW',
    interpolation: { escapeValue: false },
    ns: ['common', 'auth', 'home', 'careLog', 'health', 'emergency'],
    defaultNS: 'common',
  });

export default i18n;
```

### 4.6 自動翻譯橋梁（照護日誌）

照服員用印尼語填寫 → 後端自動翻譯 → 家屬以中文閱讀：

```typescript
// services/translation.service.ts
import { Injectable } from '@nestjs/common';

@Injectable()
export class TranslationService {
  // 使用 Claude API 進行精準醫療/長照術語翻譯
  async translateCareLog(
    text: string,
    fromLang: string,
    toLang: string,
  ): Promise<string> {
    // 優先使用術語對照表（離線）
    const glossaryResult = this.applyGlossary(text, fromLang, toLang);
    if (glossaryResult) return glossaryResult;

    // 再呼叫 Claude API 翻譯（保留長照術語準確性）
    const response = await this.claudeClient.messages.create({
      model: 'claude-sonnet-4-6',
      max_tokens: 1024,
      system: `你是台灣長照領域的專業翻譯員。請將以下${fromLang}文字準確翻譯成${toLang}，
               保持長照術語的專業性，語氣溫暖親切。`,
      messages: [{ role: 'user', content: text }],
    });

    return response.content[0].text;
  }

  // 離線術語對照表（常用詞彙）
  private glossaryMap = {
    'id→zh-TW': {
      'membantu mandi': '協助沐浴',
      'menyiapkan makan': '準備餐食',
      'jatuh': '跌倒',
      'demam': '發燒',
      'minum obat': '服藥',
    },
    'zh-TW→id': {
      '協助沐浴': 'membantu mandi',
      '跌倒': 'jatuh',
      '發燒': 'demam',
    }
  };
}
```

## 八、品質保證

### 8.1 翻譯品質流程

```
開發者新增中文文字
    ↓
機器預翻譯（DeepL API / Claude API）
    ↓
在台印尼籍校對員人工審查
    ↓
長照領域諮詢（確認術語正確）
    ↓
上線 A/B 測試（印尼籍使用者回饋）
    ↓
定期更新與維護
```

### 8.2 校對員招募

- 與各地移工服務站合作，招募具雙語能力的移工協助校對
- 每次校對提供小額報酬（Google Play 點數或現金）
- 目標：建立社區共創的翻譯機制

### 8.3 翻譯缺漏處理

```typescript
// 翻譯缺漏時的 Fallback 策略
i18n.on('missingKey', (lngs, namespace, key) => {
  // 1. 先嘗試 fallback 語言（zh-TW）
  // 2. 記錄缺漏 key 到後端，供下次更新補充
  logMissingTranslation({ lngs, namespace, key });
});
```
