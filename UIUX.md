# AnTsiSeCare - UI/UX 設計規範

---

## 一、設計原則

### 1.1 核心設計價值

| 原則 | 說明 |
|------|------|
| **溫暖** | 色調溫暖親切，避免冷硬的醫療感 |
| **簡單** | 操作步驟最少化，適合高齡者使用 |
| **安全** | 緊急功能永遠可及，重要操作有確認 |
| **包容** | 支援大字體、語音、低視力輔助 |
| **在地** | 客語支援，全台在地元素融入設計 |

## 五、無障礙設計規範

### 5.1 視覺無障礙

```
對比度：
  正常文字：≥ 4.5:1（WCAG AA）
  大文字（18px+）：≥ 3:1
  互動元素：≥ 3:1

色彩：不以顏色作為唯一資訊傳達方式（配合圖示）
焦點指示：清晰的 Focus Ring（3px Primary-500）
```

### 5.2 操作無障礙

```
觸控目標：最小 44×44px（Apple HIG）/ 48×48dp（Material）
手勢：避免複雜多指手勢，提供替代操作
捲動：避免橫向捲動
```

### 5.3 螢幕閱讀器支援

```
React Native：
  accessibilityLabel（所有互動元素）
  accessibilityHint（說明動作結果）
  accessibilityRole（button, link, header...）
  accessibilityState（disabled, selected...）

SOS 按鈕：
  accessibilityLabel="緊急 SOS 按鈕，長按三秒呼叫緊急聯絡人"
```

### 5.4 長者模式切換

```
設定 > 顯示 > 長者模式（開啟後）：
  ✓ 字體放大 1.25 倍
  ✓ 按鈕加大 1.15 倍
  ✓ 簡化導航（只顯示核心 3 個 Tab）
  ✓ 動畫關閉（減少視覺干擾）
  ✓ 震動回饋加強
```

## 六、多語言介面支援

### 6.1 語言切換 UI

```
設定頁 > 語言（Language / Bahasa / 語言）：

  🇹🇼 繁體中文    ← 預設
  🇮🇩 Bahasa Indonesia  ← 第一優先擴充
  🇬🇧 English
  （客語 - 語音輔助）

  裝置語言自動偵測：
    印尼系統語言 → 自動切換印尼語介面
```

### 6.2 印尼語（Bahasa Indonesia）介面設計要點

```
文字方向：LTR（與中文相同，無 RTL 問題）

圖示優先原則：
  - 服務項目使用圖示 + 文字，降低語言依賴
  - 勾選式日誌（checkbox list）減少自由輸入需求

語音輸入支援：
  - 印尼語語音輸入（Google Speech-to-Text）
  - 適合識字率較低或打字不便的使用者

文化色彩考量：
  - 綠色（#2E7D6B）在印尼文化中象徵伊斯蘭與自然，符合
  - 紅色（SOS）在兩種文化中皆代表緊急，一致

禱告時間（選用功能 Waktu Salat）：
  - 印尼籍看護可開啟「禱告提醒」
  - 基於裝置 GPS 自動計算當地禱告時間
  - 純選用，不影響主要功能
```

### 6.3 客語（Hakka）介面支援

```
觸發方式：長按任一按鈕 1 秒，播放客語語音說明

語音內容（客語四縣腔）：
  SOS 按鈕：「緊急求救，係毋係愛打電話分緊急聯絡人？」
  預約按鈕：「預約服務，請選擇愛預約個服務類型」
  首頁：「您好，今日有兩件事情提醒您」
```

## 七、動畫與過場設計

```
頁面切換：Slide（向左推入）250ms ease-in-out
卡片出現：Fade + SlideUp 200ms
SOS 按鈕：Pulse 動畫（1.5s 循環）
載入狀態：Skeleton Screen（骨架屏）
成功確認：綠色 Checkmark 動畫 300ms

原則：
  - 動畫時長 < 300ms（避免老年人等待焦慮）
  - 提供「關閉動畫」選項（無障礙設定）
```

## 九、AAA 高對比色票（WCAG 2.1 AAA 級）

> 針對長輩老花眼（遠視散光）與移工在弱光環境操作，所有核心色票需達 WCAG 2.1 **AAA 7:1** 對比值。

### 9.4 色盲友善輔助設計

```
原則：不以顏色作為唯一資訊傳達媒介

  ✅ 正常狀態 → 綠色圖示 + ✓ 符號 + 「正常」文字
  ⚠️ 異常狀態 → 紅色圖示 + ✕ 符號 + 「異常」文字
  🆘 緊急狀態 → 紅色背景 + ! 符號 + 脈衝邊框動畫

  色盲類型涵蓋：
    - 紅綠色盲（Deuteranopia）：以圖形 + 紋理輔助
    - 全色盲（Achromatopsia）：以高亮度對比（明暗）區分
```

### 10.2 雙語即時切換按鈕規格

```
位置：頁面右上角（導覽列右側），固定懸浮

  ┌──────────────┐
  │ 🌐 中 / id  │  ← 顯示當前語言，點擊切換
  └──────────────┘
  - 寬度：80dp，高度：36dp
  - 圓角：18dp（膠囊型）
  - 背景：Primary-100 #E8F5F1
  - 邊框：Primary-500 #2E7D6B  1px
  - 文字：Gray-900 #212121  14px Bold
  - 切換動畫：Cross-fade 200ms

行為：
  - 點擊後所有可見文字立即切換語言
  - 語言偏好儲存至 Hive 本機（不需重啟）
  - 切換範圍：紀錄內容、標籤、按鈕文字
  - 不切換：長者姓名、藥品名（固定顯示中文）
```

### 10.3 圖示狀態交互規格

```
每個照護圖示的狀態機（State Machine）：

  [未執行]  灰色，無勾選 ─
      ↓ 點擊
  [執行中]  主色框線，顯示計時器
      ↓ 完成
  [正常]    綠色底，✓ 符號，可展開備註
      ↓ 發現異常
  [異常]    紅色底，✕ 符號，強制輸入備註（≥10字）
      ↓ 備註完成
  [異常已記錄]  紅色底 + 📝 符號，已傳通知給家屬

觸控熱區：最小 64dp × 64dp（長者模式 80dp × 80dp）
圖示大小：32dp（長者模式 40dp）
標籤文字：12px（長者模式 16px Bold）
```
